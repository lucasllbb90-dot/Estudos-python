# Estudos-python
Repositório de estudos de Python: do zero à automação de sistemas reais.

Este repositório foi criado para documentar a minha jornada na programação, iniciada em fevereiro de 2026. Aqui guardo meus exercícios de lógica, desafios de cursos e projetos de automação.

## O que você vai encontrar aqui?
* **Exercícios de Lógica:** Desafios do Curso em Vídeo (Guanabara).
* **Projetos Práticos:** Ferramentas que criei para resolver problemas do dia a dia (como cálculos de orçamento e folha de pagamento).
* **Otimização:** Testes de como deixar códigos simples mais eficientes e limpos.

## Ferramentas Utilizadas
* Python 3
* Pydroid 3 (Mobile)
* VS Code (Notebook)
